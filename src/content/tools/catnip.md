The CatSniffer host toolset (CatSniffer-Tools). Its pycatsniffer drives a CatSniffer to sniff IEEE 802.15.4 / Zigbee and BLE with native Wireshark extcap integration; it also presents the CatSniffer as a virtual HCI (vHCI) Bluetooth adapter on Linux so host BLE tools like Bleak and bettercap can drive it.
